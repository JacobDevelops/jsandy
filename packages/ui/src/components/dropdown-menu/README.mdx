# DropdownMenu Component

The DropdownMenu component is a flexible and accessible solution for creating dropdown menus in React applications. It provides a comprehensive set of subcomponents to build complex menu structures with various item types, nested submenus, and keyboard navigation support.

## Features

- Customizable trigger element
- Support for standard menu items, checkbox items, and radio items
- Nested submenus for hierarchical options
- Keyboard navigation and accessibility features
- Customizable styling through className props
- Integration with design system components and utilities

## Installation

To use the DropdownMenu component in your project, install it via npm:

```bash
npm install @jsandy/ui
```

## Usage

Here's a basic example of how to use the DropdownMenu component:

```jsx
import {
  DropdownMenu,
  DropdownMenuTrigger,
  DropdownMenuContent,
  DropdownMenuItem,
} from '@jsandy/ui';
import { Button } from '@jsandy/ui';

function MyComponent() {
  return (
    <DropdownMenu>
      <DropdownMenuTrigger asChild>
        <Button variant="outline">Open Menu</Button>
      </DropdownMenuTrigger>
      <DropdownMenuContent>
        <DropdownMenuItem>New Tab</DropdownMenuItem>
        <DropdownMenuItem>New Window</DropdownMenuItem>
        <DropdownMenuItem>Settings</DropdownMenuItem>
      </DropdownMenuContent>
    </DropdownMenu>
  );
}
```

## Advanced Usage

### Checkbox Items

You can create menu items with checkboxes:

```jsx
<DropdownMenuContent>
  <DropdownMenuCheckboxItem>Show Toolbar</DropdownMenuCheckboxItem>
  <DropdownMenuCheckboxItem>Show Bookmarks Bar</DropdownMenuCheckboxItem>
</DropdownMenuContent>
```

### Radio Items

For mutually exclusive options, use radio items:

```jsx
<DropdownMenuContent>
  <DropdownMenuRadioGroup value="light">
    <DropdownMenuRadioItem value="light">Light Theme</DropdownMenuRadioItem>
    <DropdownMenuRadioItem value="dark">Dark Theme</DropdownMenuRadioItem>
    <DropdownMenuRadioItem value="system">System Theme</DropdownMenuRadioItem>
  </DropdownMenuRadioGroup>
</DropdownMenuContent>
```

### Nested Submenus

Create hierarchical menus with nested submenus:

```jsx
<DropdownMenuContent>
  <DropdownMenuItem>New Tab</DropdownMenuItem>
  <DropdownMenuSub>
    <DropdownMenuSubTrigger>More Tools</DropdownMenuSubTrigger>
    <DropdownMenuPortal>
      <DropdownMenuSubContent>
        <DropdownMenuItem>Save Page As...</DropdownMenuItem>
        <DropdownMenuItem>Create Shortcut...</DropdownMenuItem>
      </DropdownMenuSubContent>
    </DropdownMenuPortal>
  </DropdownMenuSub>
</DropdownMenuContent>
```

## Accessibility

The DropdownMenu component is built with accessibility in mind:

- Proper ARIA attributes are automatically applied
- Keyboard navigation is supported (arrow keys, Enter, Space, Esc)
- Focus management is handled automatically

## Customization

You can customize the appearance of the DropdownMenu and its subcomponents using the `className` prop. The component uses CSS modules, so you can override styles by targeting the appropriate class names.

## Best Practices

1. Use clear and concise labels for menu items
2. Group related items together and use separators to improve readability
3. Limit the number of items in a single menu to avoid overwhelming users
4. Use submenus for organizing complex hierarchies, but try not to nest too deeply
5. Provide keyboard shortcuts for frequently used actions
