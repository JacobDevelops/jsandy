# Pagination Component

## Overview

The Pagination component is a flexible and accessible solution for navigating through multiple pages of content in React applications. It provides a set of composable components that can be used to create custom pagination layouts.

## Features

- Fully customizable appearance
- Accessible, with proper ARIA attributes
- Responsive design
- Support for previous/next navigation
- Ellipsis for indicating additional pages
- Active page highlighting

## Installation

To use the Pagination component, ensure you have the necessary dependencies installed in your project:

```bash
npm install @jsandy/ui
```

## Usage

Here's a basic example of how to use the Pagination component:

```jsx
import {
  Pagination,
  PaginationContent,
  PaginationItem,
  PaginationLink,
  PaginationPrevious,
  PaginationNext,
  PaginationEllipsis,
} from '@jsandy/ui/pagination';

function MyPagination() {
  return (
    <Pagination>
      <PaginationContent>
        <PaginationItem>
          <PaginationPrevious href="#" />
        </PaginationItem>
        <PaginationItem>
          <PaginationLink href="#">1</PaginationLink>
        </PaginationItem>
        <PaginationItem>
          <PaginationLink href="#" isActive>
            2
          </PaginationLink>
        </PaginationItem>
        <PaginationItem>
          <PaginationLink href="#">3</PaginationLink>
        </PaginationItem>
        <PaginationItem>
          <PaginationEllipsis />
        </PaginationItem>
        <PaginationItem>
          <PaginationNext href="#" />
        </PaginationItem>
      </PaginationContent>
    </Pagination>
  );
}
```

## Components

### Pagination

The root component that wraps all pagination elements.

**Props:**

- Accepts all props of a `<nav>` element

### PaginationContent

A container for pagination items.

**Props:**

- Accepts all props of a `<ul>` element

### PaginationItem

A wrapper for individual pagination elements.

**Props:**

- Accepts all props of a `<li>` element

### PaginationLink

A component for rendering page links.

**Props:**

- `isActive`: boolean - Indicates if the link represents the current page
- `size`: string - Size of the link (default: 'icon')
- Accepts all props of an `<a>` element

### PaginationPrevious

A component for the "Previous" navigation link.

**Props:**

- Accepts all props of `PaginationLink`

### PaginationNext

A component for the "Next" navigation link.

**Props:**

- Accepts all props of `PaginationLink`

### PaginationEllipsis

A component for indicating additional pages.

**Props:**

- Accepts all props of a `<span>` element

## Customization

The Pagination component uses Tailwind CSS classes for styling. You can customize its appearance by:

1. Passing additional classes via the `className` prop on any component.
2. Modifying the `buttonVariants` function in your project's theme configuration.
3. Overriding the default styles in your CSS.

## Accessibility

The Pagination component is built with accessibility in mind:

- It uses semantic HTML elements (`<nav>`, `<ul>`, `<li>`)
- It includes proper ARIA labels and roles
- It supports keyboard navigation
